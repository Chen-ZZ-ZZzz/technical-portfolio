# Collatz Sequence Calculator

This small Python program calculates the **Collatz sequence** (also known as the 3n + 1 problem) for a given integer input until it reaches 1.

## Purpose
- Demonstrates basic Python control flow, loops, and conditional logic.
- Handles user input and simple error checking.

## How to Use
1. Run the script:
   ```bash
   python collatz_sequence.py
2. Enter an integer number when prompted.
3. The program will print the Collatz sequence for that number until it reaches 1.
